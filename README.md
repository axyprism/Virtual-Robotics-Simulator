# Virtual Robotics Simulator
[![Itch.io](https://img.shields.io/badge/itch.io-%23FF0B34.svg?logo=Itch.io&logoColor=white)](https://axyprism.itch.io/virtual-robotics-simulator)

<img width="719" height="478" alt="SCR-20260627-prey" src="https://github.com/user-attachments/assets/7d669e54-1459-4c30-bf6b-f39ee001c2ac" />

This is a game where you can play FRC (First Robotics Competition) games and participate in virtual competitions. 
Currently, you can spawn in a simple swerve robot and control it, as well as play with other people over LAN. 

You can find a release in the releases tab... but I reccomend you get the game (and any updates) from Itch since I will probably be updating it more often there.
The game is available for Mac and Windows. Controls are listed on Itch.

## Background
I made this game as I am on a FRC team and have a lot of intrest in robotics, as well as making games. I wanted to let people experience FRC competitions with not just support for many devices but also multiplayer.
